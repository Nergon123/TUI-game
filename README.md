very WIP

![Lmao](imgs/demo2.png)

`e` - player\
`T` - sign\
`W` - enemy\
`#` - walls

![sign example](imgs/sign.png)

## controls
"w" + enter: up\
"s" + enter: down\
"a" + enter: left\
"d" + enter: right

## build and run
```shell
cmake CMakeLists.txt && make run
```
